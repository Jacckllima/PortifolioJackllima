## 🕹️ Funcionalidades Premium e Interatividade

O portfólio foi totalmente projetado com micro-animações, foco em experiência do usuário e interações fluidas:

*   **💻 Terminal do Desenvolvedor Interativo (VS Code Feel)**: Uma simulação em tempo real do editor Visual Studio Code. Recrutadores podem alternar as abas ou clicar na árvore de arquivos lateral para navegar pelas minhas informações formatadas com *sintaxe colorida profissional*.
*   **📍 Linha do Tempo Dinâmica ("Minha Jornada")**: Uma representação vertical da minha jornada de aprendizado que narra a transição estruturada de conhecimentos de Infra/Redes para Front-end, Back-end e Inteligência Artificial. Totalmente adaptável para visualização em pilha única no mobile.
*   **📖 Gavetas de Estudos de Caso nos Projetos**: Em vez de apenas exibir links genéricos, cada projeto possui um botão expansível de "Estudo de Caso" que revela, em uma gaveta animada, os bastidores de desenvolvimento: **Desafio**, **Solução** e **Aprendizados**.
*   **✨ Efeito Ambient Mouse Glow (Cursor Tracker)**: Um degradê de luz suave que segue a posição física do cursor do mouse, dando profundidade e modernidade ao plano de fundo.
*   **🔍 Filtro de Habilidades Dinâmico**: Organiza as minhas 17 competências acadêmicas em badges modernos de vidro (*glassmorphism*), permitindo filtrar por categoria (Front-end, Back-end, Infraestrutura, IA) em tempo real sem recarregar a página.
*   **📱 Responsividade 100% Blindada**: Totalmente adaptado para todas as resoluções de tela, desde monitores UltraWide até dispositivos ultra-pequenos e dobráveis como o Galaxy Fold (320px).

## 🌐 Como Visualizar e Rodar Localmente

O portfólio é estático, o que significa que pode ser executado diretamente no navegador sem a necessidade de compilação ou processos de build.

### Visualização Online
Acesse o portfólio hospedado diretamente através do GitHub Pages:
👉 **[Portfólio Online da Jaqueline Lima](https://jacckllima.github.io/PortifolioJackllima/)**

### Execução em Máquina Local
Se preferir clonar e executar o projeto em sua máquina:
```bash
# 1. Clone este repositório
git clone https://github.com/Jacckllima/PortifolioJackllima.git

# 2. Navegue até a pasta do projeto
cd PortifolioJackllima

# 3. Abra o arquivo index.html no seu navegador de preferência
# No Windows/PowerShell:
Start-Process "index.html"
# Ou inicialize um servidor estático local:
npx serve .
```
